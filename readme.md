# Telzenith Landing Page
This repository contains the source code for the landing page of Telzenith, a modern Minecraft survival server hosted by [The Xeriscape Network](https://xeriscape.network/).

The site is hosted using GitHub Pages and is accessible directly through the [telzenith.xyz](https://telzenith.xyz/) domain. It serves as a static entry point for players who visit the server address via a web browser, providing an overview of the server's features, community resources, and information for connecting to the server.
